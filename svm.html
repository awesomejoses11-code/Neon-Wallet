Import Wallet

One mnemonic can hold multiple wallets (Phantom/Solflare/Backpack call these "accounts"). Index 0 is the default. If you don't see the address you expect, click "Preview addresses" to see and pick the right one.

This wallet only holds Solana assets. If your mnemonic turns out to be an Ethereum/EVM seed, this shows the address so you can confirm it and go use it in an EVM wallet (MetaMask, Rabby, etc.) — importing it here would not give you access to those funds.

A raw private key has no recovery phrase backing it up — if you lose it, this wallet can't help you recover it. Only paste keys on a device you trust.
